These data correspond to the project "Online and offline mechanisms of memory integration over development" (James, E. & Unger, L; University of York). 

This folder represents the preprocessed dataset and script for one set of preregistered analyses, released for the purposes of the Ninth Summer School on Statistical Methods for Linguistics and Psychology.

The full dataset will be released on the OSF in due course (https://osf.io/v2jbq/)
